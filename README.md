Название проекта: 
Система анализа влияния новостного потока агрегатора Smart-Lab на волатильность российских акций.

Цель:
Разработать программный комплекс для оценки корреляции между эмоциональной окраской публикаций в Telegram-канале Smart-Lab и изменениями котировок на бирже.

Задачи:
+Реализовать сбор постов и метаданных из Telegram-канала.

Разработать алгоритм идентификации эмитентов, который распознает как полные названия компаний, так и их биржевые тикеры в тексте.

Провести классификацию тональности текстов с учётом финансового контекста.

Сопоставить временные метки постов с графиками котировок акций.

Рассчитать коэффициент корреляции и визуализировать зависимости настроений от ценовых движений.

Краткое описание:
Проект представляет собой инструмент для анализа на фондовом рынке. Система собирает новости, очищает их от информационного шума и сопоставляет эмоциональный фон с реальными рыночными данными.
Результат работы — отчет о том, новости про какие акции вызывают наибольшую реакцию инвесторов.
