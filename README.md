Название проекта: 
Прогнозирование эмоционального отклика социума на финансовые публикации и его статистическая связь с динамикой фондового рынка.

Цель:
Разработать исследовательскую модель для предсказания возникновения эмоциональной реакции инвесторов (социума) на текстовые публикации канала Smart-Lab и оценки её влияния на рыночные показатели.

Задачи:
Реализовать сбор постов и метаданных из Telegram-канала.

Разработать алгоритм сопоставления текстовых упоминаний с конкретными эмитентами.

Провести классификацию текстов по признакам эмоциональной окраски. В рамках учебной задачи акцент делается на предсказании потенциальной реакции социума на текст.

Сопоставить временные метки постов с изменениями котировок акций в качестве возможных материализованных результатов социальной реакции.

Определить статистическую связь между предсказанным откликом и реальной рыночной волатильностью с помощью коэффициента корреляции и визуализации зависимости настроений от ценовых движений.

Краткое описание:
В основе проекта лежит гипотеза о финансовом тексте как о триггере, формирующем паттерны коллективного поведения на фондовом рынке. 
Результатом будет являться установление признаков корреляции, а, следовательно, и оценить необходимость проведения более детальных исследований взаимосвязи между информационным стимулом и реакцией социума.
